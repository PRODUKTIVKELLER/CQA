CQA - Code Quality Assurance for Unity

CQA is an open-source tool for keeping your code base consistent by using rules that are executed on your code.

CQA comes with a set of predefined rules to get you started quickly. This includes metric checks like:

  • No class has more than 300 lines of code.

There are also more powerful rules like:

  • The maximum inheritance depth is two.

New rules are written in Cypher, the query language which is used in the graph database neo4j.

Getting Started

To get started have a look at the HANDBOOK.md.

How does it work?

We use jQAssistant and the C# jQAssistant plugin to parse your codebase into a neo4j graph database. Then we use jQAssistant to run rules written in Cypher against the database. The result is a HTML report that shows which rules are fulfilled.
